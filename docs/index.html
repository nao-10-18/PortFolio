    作品(4)「収集ゲーム」

      　
    • 作品内容
      • 　Unityを使った初めての作品で収集型のアクションゲームを制作しました。
      　一定数のスコアを獲得してゴールすることでクリアとなります。スコアを獲得するにはオブジェクトにふれることで、スコアがプラスされたりマイナスされたりします。
      　また、自身がフィールドから落下してもスコアがマイナスされます。加えてオブジェクトを触れると消えてしまうので、
      　スコアが一定値にならない場合も想定して、そうなった場合のリスタートも導入しています。
      　また、ギミックとして触れてはいけないオブジェクトも導入してます。

    作品(5)「群青」

      　
    • 作品内容
      • 　入学して初めて作成した作品で、3種類のジャンルをミニゲーム形式で作りました。
      　制作にあたって出題されたテーマは、「さんすくみ」要素を取り入れることでした。
      　私はシナリオがあったほうが取り組みやすく感じているので、「自分と向き合うこと」を個人の中でテーマとしてスポーツなどでいう
      　「心・技・体」をそれぞれのゲームとして落とし込み、さんすくみ要素を含んだゲーム制作をしました。
      　タイトルの群青は現在のタイトル画面の背景色が白ですが、以下のゲームをクリアすることで背景に色をつけるよう意味を込めて制作しました。
      　
      　

      　まずは、「体」を表したRPGの戦闘型ゲームです。
      　ぐー、ちょき、ぱーのさんすくみをシンプルに再現しており戦う選択後にぐー、ちょき、ぱーを選んで相手とじゃんけんをした結果によって、ダメージを受けたります。
      　また、自身のHPは、数値としてではなくシステム部分でどういう状態かを表示させるようにしてます。

      　次に、「技」を表したパックマンのような収集型ゲームです。
      　フィールドに設置されたオーブを取ることで自身のパワーがプラスされたりマイナスされたりし、
      　じゃんけんの手を指の本数で表現してます。（ぐーの状態は指の本数が折りたたまれている状態なので、パワー:0 = グー）
      　その状態で自分の(じゃんけんの手で)勝てる相手にふれることで、敵(じゃんけんの手の画像)を減らし、
      　また(じゃんけんの手で)負ける敵に触れてしまうと、自分がやられてしまうゲームとなっております。
      　敵も自分が有利な場合は、プレイヤーに向かってきたり逃げたりします。
      　ぐー、ちょき、ぱーのそれぞれの敵を倒すとクリアとなります。

      　最後に、「心」を表したシューティングゲームです。
      　プレイヤーである黄色の球体は、相手の色を見て下記図さんすくみ図の通りに弾の切り替えを行い攻撃します。
      　そして、相手は自身と同じ黄色の弾を撃って攻撃し残りのHP割合で色が赤、青、緑色に変わるようになってます。
      　自分(黄色)が自分の心を傷つけているイメージで制作しましたので倒すことを目標としておらず、相手のHP割合で正解へのヒントを表示するようにしてます。
      　
    • 操作方法
      • 　十字キー　　　　:　移動
      　エンターキー　　:　決定
      　バックスペース　:　戻る
      　
    • 制作概要
      • 　制作人数：1人 　
      　製作時期：R2.6月頃～R2.9月頃
      　使用言語：C・C++
      　開発環境：Windows10・Visual Studio 2019
      　使用ライブラリ：Dxlib
      　コード： 群青のGithub
